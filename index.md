# Froth

Froth is a programming language for microcontrollers that turns them into fully interactive computers.

Froth is weird. That's kind of the point. Froth needs to be weird to be powerful.

I suggest you take a look at "getting started", where you can download the VSCode plugin and the Froth CLI, as
well as take your first baby steps with the language.

Froth is based on Forth, an old language that pioneered the concept of "interaction" in computers: the at-the-time
radical idea that you could change the behavior of a system without having to fully reset it.

At first sight, Froth might seem weird and mind-bending. I've put together tutorials and guides that teach you the language
step by step - the Froth language is extremely simple, but very powerful. It only takes a few days to become fully comfortable
writing complex Froth languages. It's also fun (if you're into that sort of thing) to learn how to think about programming
in a different way, one that is both closer to the "metal" of the hardware, but also far more flexible and fun than what
you're probably used to.
